# input()函数的工作原理
input()函数让程序暂停运行，等待用户输入一些文本。获得用户输入后，Python将其赋给一个变量，以便使用。
```python
message = input("请输入你的名字: ")
print(message)
```
输出结如下：
- ![](assets/Python基础之%20用户输入%20和while循环/file-20260118004234454.png)
- input()函数接受一个参数，即要向用户显示的提示（prompt），让用户知道输入什么样的内容。在这个例子中，当python运行第一行代码时，用户将看到提示：”请输入你的名字“。程序等待用户输入，并在用户回车后继续运行。
# 编写清晰的提示词
每当用户input()函数时，都应该指定清晰易懂的提示，准确地指出希望用户提供什么样的信息--，有时候，提示可能超过一行。例如，你可能需要指出获取特定输入的原因。在这种情况下，可先将提示赋给一个变量，在将这个变量传递给input()函数。这样，即便提示超过一行，input（）语句也会非常清晰。
```python
prompt = "如果你愿意分享你的姓名,请你打出你的名字,如果不愿意分享,"
prompt += "请输入你的别名名: "
messing = input(prompt)
print(messing)

```
这个示例演示了一种创建多行字符串的方式。第一行将消息的前半部分赋给变量prompt。在第二行中，运算符+=在赋给变量prompt的字符串末尾追加一个字符串。
最终的提示占两行，且冒号后面有一个空格，这也使为了使其更加清晰：

# 使用 int() 来获取数值输入

在使用input（）函数时，Python会将用户输入解读为字符串。看下面的示例
- ![](assets/Python基础之%20用户输入%20和while循环/file-20260118224742485.png)
- 用户输入的是数22，但是我们请求Python提供变量的值时，它返回的是‘22’，用户输入的是数值但现在是字符串表示。我们怎么知道这个Python将输入解读成了字符串呢？因为这个数字是用引号引起了的。如果只想打印输入，这一点问题都没有：但如果试图将输入的数进行数学运算，就会出错。如下所示
- ![](assets/Python基础之%20用户输入%20和while循环/file-20260118225333677.png)
- 试图将该输入用于数值比较时，Python会报错，因为它无法将字符串和整数进行比较：不能将赋给age的字符串“22”与数值25进行比较。
- 为了解决这个问题，可使用int()函数将输入的字符串转换为数值，确保能够成功地执行比较操作：
- ![](assets/Python基础之%20用户输入%20和while循环/file-20260118225826486.png)
- 如何在实际的中使用int()函数呢？请看下面的程序
```python
age = input("请输入你的年龄: ")
age = int(age)
if age >= 18:
    print('年轻人加油干')
else:
    print("小朋友,要好号学习哦")
    
```
